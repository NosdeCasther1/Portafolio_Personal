## Sobre el Proyecto

Este sistema fue diseñado para optimizar el proceso de preventa en empresas de seguridad electrónica. Permite a los técnicos y asesores comerciales generar cotizaciones profesionales de manera rápida, integrando catálogos de productos y cálculos automáticos de mano de obra y materiales.

### Características Principales

- **Gestión de Catálogo:** Administración centralizada de cámaras, grabadores y accesorios.
- **Generación de PDF:** Exportación inmediata de cotizaciones con diseño corporativo.
- **Cálculos Automáticos:** Desglose preciso de costos, márgenes e impuestos.
- **Interfaz Intuitiva:** Desarrollado con Next.js y Tailwind CSS para una experiencia fluida y responsive.
