# Bienvenido a mi primer proyecto

Este es un ejemplo de cómo se ve un archivo MDX. Puedes escribir **Markdown** normal y el sistema lo renderizará correctamente.

## Características principales

- Renderizado ultra rápido con Next.js.
- Contenido gestionado localmente a través de archivos `.mdx`.
- Frontmatter para metadatos como el título y la fecha.

### Código de ejemplo

```typescript
const saludo = "Hola Mundo";
console.log(saludo);
```

¡Espero que este sistema te sea de gran utilidad para tu portafolio!
