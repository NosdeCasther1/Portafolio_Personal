## Sobre el Proyecto

OptiBill ISP es una solución integral para proveedores de servicios de internet que permite gestionar clientes, planes de velocidad, facturación recurrente, cortes automáticos y reportes de consumo. El núcleo técnico de este proyecto se enfocó en la auditoría, optimización y benchmarking de consultas SQL en MySQL para garantizar tiempos de respuesta inferiores a 200ms en bases de datos con miles de registros.
