## Sobre el Proyecto

PlagaControl es un software SaaS diseñado para la administración operativa de empresas dedicadas al control de plagas y saneamiento ambiental. El sistema permite agendar visitas, asignar técnicos de campo, documentar las aplicaciones químicas (cumpliendo con normativas de salud) y facturar el servicio directamente desde la interfaz del usuario.
