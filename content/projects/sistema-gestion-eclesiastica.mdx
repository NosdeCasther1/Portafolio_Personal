## Sobre el Proyecto

Este sistema fue desarrollado para digitalizar y automatizar los procesos operativos y administrativos de una organización eclesiástica. La plataforma proporciona una solución centralizada y segura para la gestión del personal, control de eventos y la comunicación interna, optimizando las actividades diarias de la organización.
