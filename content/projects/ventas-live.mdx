## Ventas en Tiempo Real

Ventas Live es una solución diseñada para emprendedores que utilizan redes sociales para vender productos en vivo. El sistema permite registrar pedidos, gestionar el inventario disponible y confirmar pagos de forma dinámica sin interrumpir la transmisión.

### Funcionalidades Clave

- **Dashboard en Vivo:** Panel interactivo para el vendedor con actualizaciones instantáneas.
- **Gestión de Stock:** Control automático de unidades disponibles para evitar sobreventas.
- **Backend Robusto:** Implementación con Node.js para manejar múltiples conexiones simultáneas.
- **Persistencia de Datos:** Almacenamiento eficiente de transacciones y clientes para seguimiento post-venta.
